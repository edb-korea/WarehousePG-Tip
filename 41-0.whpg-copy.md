# WarehousePG 데이터 이관 작업 절차 (gpadmin → targetdb)

## 1. Source DB 스키마 덤프 (gpadmin)

```bash
# gpadmin 데이터베이스 스키마만 덤프
pg_dump -Cs -U gpadmin -d gpadmin -f ddl_gpadmin.sql

# 전체 클러스터 스키마 덤프 (역할/전역 객체 포함)
pg_dumpall -Cs -U gpadmin -d gpadmin -f ddl_gpadmin.sql

# tpcds 데이터베이스 스키마 덤프
pg_dump -Cs tpcds -f ddl_tpcds.sql
```

## 2. Target DB 생성

```bash
# targetdb 재생성
dropdb --if-exists targetdb
createdb targetdb

# tpcds_dst 재생성
dropdb --if-exists tpcds_dst
createdb tpcds_dst
```

## 3. DDL 수정

```text
1.ddl_gpadmin.sql 파일 내 데이터베이스명을 gpadmin → targetdb 로 수정
2.테이블 분산키 조정
3.테이블 압축 변경. quicklz -> zstd
```

## 4. Target DB에 스키마 생성

```bash
psql -U gpadmin -d targetdb -f ddl_gpadmin.sql
psql -d tpcds -f ddl_tpcds.sql
```

## 5. whpg-copy를 이용한 데이터 이관

### 5-1. 연결 상태 점검 (diagnose)

```bash
whpg-copy diagnose \
  -s postgres://gpadmin@whpg-m:5432/tpcds \
  -d postgres://gpadmin@cdw:5432/tpcds
```

### 5-2. 데이터 복사 (copy)

```bash
whpg-copy copy \
  -s postgres://gpadmin@whpg-m:5432 \
  -d postgres://gpadmin@whpg-m:5432 \
  --src-db targetdb \
  --dst-db tpcds_dst \
  --target-mode append \
  --workers 4 \
  --compression false \
  --validate-method count
```

### 5-3. 이관 대상 테이블 목록 추출 (파티션 자식 테이블 제외)

```sql
SELECT '"' || n.nspname || '.' || c.relname || '",' AS table_name
FROM pg_class c
JOIN pg_namespace n ON n.oid = c.relnamespace
WHERE c.relkind = 'r'
  AND n.nspname NOT IN ('pg_catalog', 'information_schema')
  -- 파티션 자식 테이블 제외 (pg_inherits에 자식으로 등록된 것들)
  AND c.oid NOT IN (
      SELECT inhrelid FROM pg_inherits
  )
ORDER BY n.nspname, c.relname;
```

### 5-4. 이관 대상 테이블 목록 추출 (전체 테이블)

```sql
SELECT '"' || schemaname || '.' || tablename || '",' AS table_name
FROM pg_tables
WHERE schemaname NOT IN ('pg_catalog', 'information_schema')
ORDER BY schemaname, tablename;
```
